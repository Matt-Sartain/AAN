Uploading Links for All Publishers
When creating, updating or deleting links for All Publishers, the following fields should be used in this order: Start Date (if applicable), End Date (if applicable), Link Name, Landing Page URL, Banner URL, Link Copy, Link ID, Promotion Type).
Start Date Optional for Creating or Updating Links
Format MM/DD/YYYY. If empty, will default to Today.
End Date Optional for Creating or Updating Links
Format MM/DD/YYYY. If empty, will default to Ongoing.
Link Name: Required for Creating and Updating links.
Note: This field has a max of 100 characters.
Landing Page URL Required for Creating and Updating links; must be a valid URL.
Banner URL Optional for Creating and Updating links; must be a valid URL.
Link Copy Required for Creating Text links; optional for Updating Text Links. Note: This field has a max of 255 characters
Link ID: Required for Updating or Deleting links.
Promotion Type Optional for Creating and Updating links. Accepted Promotion Types are: Coupon, Gift with Purchase, Holiday, Hot Product, Purchase with Purchase, Sale, Shipping, Sweepstakes.
Uploading Links for Individual Publishers
When creating, updating or deleting links for Individual Publishers, the following fields should be used in this order: Link Name, Publisher ID, Landing Page URL, Banner URL, Link Copy, Link ID).
Link Name: Required for Creating and Updating links. Note: This field has a max of 100 characters.
Publisher ID: Required for Creating and Updating links; must be a valid Publisher Web Site ID
Landing Page URL Required for Creating and Updating links; must be a valid URL.
Banner URL Optional for Creating and Updating links; must be a valid URL.
Link Copy Required for Creating Text links; optional for Updating Text Links.
Note: This field has a max of 255 characters
Link ID: Required for Updating or Deleting links.
