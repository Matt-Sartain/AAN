Managing Links and Banners

Atrinsic Affiliate Network allows you to add links individually and in batches by downloading templates located in the Help bar under Upload Multiple.

When uploading batches of banners you must choose to host the images or to use the Upload Images and have them hosted on the Atrinsic Affiliate Network.

Links

The main menu when looking to managing your links shows the link type, the count and provides buttons to add or edit the specific link type within the network. Clicking the link type will take you to a list of the current links of that type you have loaded into the network. If you find that you are at the maximum allowed amount of a specific link you can contact your account manager to have this number changed.

From the link type list view you are able to edit specific links (landing page URL, text, image URL, etc.) or delete the link itself from the network.

Best Practice Tip: If a specific link or banner is no longer to be used when promoting your offer it is best to change the wording and/or replace the image with the new banner of the same size so as not to create dead links or broken images on your partners page.

Uploading Multiple Images

You can upload a zip file to upload multiple images at a time to the server or you may upload one image at a time.

File types must be jpeg, .jpg, .gif or .png and less than 200 KB in file size.

Advertisers are allowed a maximum of 50 images for each account. Please contact your account manager should you need more.

If you are uploading a zip file, use the browse button to locate and select the file. Enter an email address to receive a confirmation email when your upload has completed. Once you have entered your confirmation email, click "Upload".

You will receive a confirmation email with a list of the image urls once the upload is complete.

Note that you may not delete an image if it is associated with an active link within the interface.

Uploading Links for All Publishers

When creating, updating or deleting links for All Publishers, the following fields should be used in this order: Start Date (if applicable), End Date (if applicable), Link Name, Landing Page URL, Banner URL, Link Copy, Link ID, Promotion Type).

Start Date Optional for Creating or Updating Links Format MM/DD/YYYY. If empty, will default to Today.
End Date Optional for Creating or Updating Links Format MM/DD/YYYY. If empty, will default to Ongoing.
Link Name Required for Creating and Updating links. Note: This field has a max of 100 characters.
Landing Page URL Required for Creating and Updating links; must be a valid URL.
Banner URL Optional for Creating and Updating links; must be a valid URL. This information will be available from the confirmation email you receive from the Atrinsic Affiliate Network when hosting multiple images as described above.
Link Copy Required for Creating Text links; optional for Updating Text Links. Note: This field has a max of 255 characters
Link ID Required for Updating or Deleting links.
Promotion Type Optional for Creating and Updating links. Accepted Promotion Types are: Coupon, Gift with Purchase, Holiday, Hot Product, Purchase with Purchase, Sale, Shipping, Sweepstakes.
