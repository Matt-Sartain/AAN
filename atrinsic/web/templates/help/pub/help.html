Introduction

Purpose

Welcome to the Atrinsic Affiliate Network. This guide is meant to function as a supplemental piece of information to be used along with your Atrinsic Affiliate Network point of contact. At any time you can refer to this document or contact your point of contact for help with the network.

Scope

This document is limited to the basic and most commonly used functions of the Atrinsisc Interactive Network from the Publishers dashboard. If this document doesn’t address your specific concerns please contact your network point of contact.

Customizing your Dashboard

The Atrinsic Affiliate Network allows you to customize the dashboard with different reporting widgets enabling you to get a quick overview of what matters to you simply by logging in. Beneath the Network logo on your dashboard, you can “Add a widget...”

Hover over the options to see a preview and then select the reporting you would like to add to your dashboard. When the new widget appears you can customize the view of the output by clicking the plus icon in the lower right hand corner (IMAGE 03), and re-order your widgets by dragging them on screen.

Retrieving Links and Banners

You can easily access the available links of all the programs you’ve joined in two ways. The first is to simply click the “get links” icon on the quick launch bar in the lower left of your screen [IMAGE 08].

Additionally you can access this page from the main navigation bar by clicking “Links”. From the Links screen you’ll be able to filter the available links by type, vertical, and promotion type. Links will appear as program badges and selecting the link icon allows you to obtain the HTML needed for your page.

Checking Network Messages

Messages from your Advertisers and announcements from the Network itself can be found in your Network inbox. You can easily see when you have new images based on the message icon located on the main navigation bar in the upper right hand corner [IMAGE 11].

Clicking the message icon or clicking Messages in the main navigation bar will take you to your inbox where you can read & compose your inner Network messages. Additionally, you can change the settings for receiving notifications to your on file email address by clicking settings on the sub menu bar.

Inquiries

Payment and Order inquiries can be sent from this menu in specific relation to a partnership. This information is then reviewed by an Atrinsic Affiliate Network admin. All such inquiries in relation to payments and orders should be submitted from this sub menu of the Message center.

Settings

Message settings allows you to decide what types of messages the Atrinsic Affiliate Network will send to which of your associated email accounts. In addition to being able to filter specific message types to specific email accounts you can also set a primary contact email address here.

Running Reports

The Atrinsic Affiliate Network has built in reporting that allows you to track the progress of your Advertiser relationships. Reporting is accessed by clicking Reports on the main navigation bar.

Reports can be further broken down by time frame, advertiser, and vertical. You will need to select a start and end date for the report before running the report. Select a report type and run report.

Once a report has been run the columns can be sorted easily by clicking on a column header. Furthermore reports can be downloaded, or changed between three separate views by accessing the report menu by clicking the plus sign on the bottom left of the results.

Settings

Under Settings from the Main navigation menu you are able to access a host of options to control, inform and report on your program.

All information is locked and must be edited by contacting your Atrinsic Affiliate Network point of contact.

6.1 Users

Users are where you will add the multiple program users that will be working within your program. This list serves as a contact list for the network when sending updates, information and alerts. One of the Users must be set as the Admin and the first user entered will be Admin by default.

6.2 Payment Info

Payment info allows you to customize how you would like to be paid. It allows you to change the currency, method, classification and tax id.

6.3 Setting Up Alerts

Alerts are a powerful tool that allows you to customize your program base to inform you of significant changes in traffic and payouts across your advertiser partnerships. Using alerts will help you to target advertisers who may need help in properly promoting their products as well as give you insight into advertisers whom you are doing a general above average job of promoting to approach about private offers.

Select Settings from the main menu, and then select Alerts from the sub menu. Click Add Alert to bring up the criteria settings for your new alert. Select what you wish to be notified about and then select the time frame over which the alert should happen.

You can add multiple alerts and all alerts will be displayed on the Dashboard with your reports when you first log in.

6.4 Web Sites

This section is dedicated to adding the multitude of URLs you may be using to promote Advertisers and generate traffic. Each site can be listed here and then selected when submitting partnership requests in order to allow an Advertiser to see what site specifically will be used to promote their products and/or services.

6.5 Incentive Site Settings

Incentive site settings allows you as a publisher to set up an SID account to better integrate the Atrinsic Affiliate Network with your incentive tracking as well as to request data transfers for UPromise integration. SID requests are submitted for review and implementation by an Atrinsic Interactive Network admin.

6.6 Data Feeds

Atrinsic Affiliate Network allows for Data Feeds to be requested and implemented. Data feeds provide you with a list of products to push traffic too. All feeds are provided as an FTP Push, FTP Pull, or HTTP Pull format and will be flat files with comma, pipe, or tab delimiters. Once a Data Feed is requested it will be reviewed and approved by Atrinsic Affiliate Network Admin.

6.7 Piggyback Pixels

Select 'Add Pixel', then, select which advertiser it's for and what type of tracking you are using. Cut and paste your pixel content and click Add Pixel again.

6.8 Real Time Box Settings

Atrinsic Affiliate Network provides a real time account of all things happening within your program. This information appears in the upper right hand corner of the Atrinsic Affiliate Network. You can change the real time ticker variable to display a myriad of information including but not limited to Impressions, Clicks, Sales, and Commission Earned.
